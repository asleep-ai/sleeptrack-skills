# Asleep API Skills Marketplace

## Reference Documentation
- [Anthropic Skills Repository](https://github.com/anthropics/skills) - Official skills examples and structure
- [Claude Code Plugin Marketplaces](https://code.claude.com/docs/en/plugin-marketplaces) - Marketplace creation and distribution

## Overview

This repository provides a skills marketplace for Asleep API integration with Claude Code. Skills enable Claude to interact with sleep tracking data, user management, and analytics through the Asleep platform.

## Implementation Approach

This is a **skills marketplace** that will contain individual skills for different Asleep API functionalities:
- Session analysis and management
- User profile management
- Sleep statistics and reporting
- Data visualization and insights

## Marketplace Structure

Skills are organized as independent plugins that can be installed individually based on user needs.
