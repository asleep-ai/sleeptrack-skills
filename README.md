# sleeptrack-skills

A skills marketplace for developers building applications with the Asleep/Sleeptrack API. These skills help Claude assist with integrating sleep tracking features, implementing API endpoints, and building sleep analysis applications.

## Installation

### Claude Code

Add this marketplace to Claude Code:

```bash
/plugin marketplace add asleep-ai/sleeptrack-skills
```

Then browse and install individual skills, or install directly:

```bash
/plugin install <skill-name>@sleeptrack-skills
```

### Prerequisites

- Claude Code CLI installed
- Developer account with Asleep/Sleeptrack platform
- Familiarity with sleep tracking API concepts

## Available Skills

Skills in this marketplace are organized by functionality:

### Core Skills
*(Skills will be added here as they are developed)*

- **API Integration** - Implement Asleep API endpoints in your application
- **Session Management** - Build features for sleep session data handling
- **User Management** - Implement user profile and account management
- **Statistics & Analytics** - Create sleep statistics and insights features

## Usage

These skills guide Claude in helping you:
- Design and implement Asleep API integrations
- Structure your sleep tracking application
- Handle authentication and API requests
- Process and analyze sleep data
- Follow best practices for sleep tracking apps
